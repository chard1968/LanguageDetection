# LanguageDetection
Uses N-Gram Language model for predicting input text and matching them thru the word bank.

##System requirements: 
```
Java 1.8
Maven 3
```
##Technology Stack:
```
Spring 4
Spring Boot,
Junit,
Mockito
HTML,CSS
```

How to run application do an 
```unix
mvn clean
```
on directory then do an
```unix
mvn spring-boot:run
```
Form can be accessed thru http://localhost:8080/form
